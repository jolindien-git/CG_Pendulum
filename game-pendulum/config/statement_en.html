The Goal

Try to keep a frictionless pendulum standing up.

  Rules

You are controlling a pendulum. Given measurement values on its state (angle and speed) you choose to apply a torque value in order to stabilize the pendulum.
The inverted pendulum swingup problem is a classic problem in the control literature. This version of the problem is adapted from Gym "Pendulum-v0".

The pendulum works as follows:
  • Every turn it takes a new command torque: -2 <= torque <= 2.
You win if:
  • You stabilize the pendulum by keeping the angle in [-1; 1] degree during 3 consecutive turns.
You lose if:
  • You mess up.

  Expert Rules

On each turn the pendulum state is computed this way:
  • The angle time derivative (the pendulum speed) is computed (given the previous state and the torque value you sent).
  • A new angle is obtained by an Euler integration step.
  • If this angle is in [-1; 1] degree during 3 consecutive turns, you win.

  Game Input

Input for one game turn
One line of 3 floats: cosTheta, sinTheta, dotTheta.
cosTheta the cosine of the pendulum angle.
sinTheta the sine of the pendulum angle.
dotTheta the time derivative of the angle.
Output for one game turn
One line : one float torque.
Constraints
-8 <= dotTheta <= 8
-2 <= torque <= 2

Response time for the first turn ≤ 1000 ms
Response time per turn ≤ 50 ms
